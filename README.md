# SPARK Airways - Flight Booking System

A modern, responsive web application for flight booking and management with a beautiful user interface and seamless user experience.

### Flight Booking System
- **Smart Flight Search**: Search flights by route, date, and passengers
- **Filter Options**: Sort by cheapest or fastest flights
- **Real-time Results**: Dynamic flight results with pricing and availability
- **Flight Details**: Comprehensive flight information including duration, timing, and seat availability
- **Responsive Design**: Works seamlessly across all devices

### User Authentication
- **Secure Login System**: Modern login interface with validation
- **User Registration**: Complete signup process with form validation
- **Password Security**: Strength indicator and secure password handling
- **Session Management**: Remember me functionality

### Additional Features
- **AI Chatbot**: Interactive flight assistant for customer support
- **Beautiful UI/UX**: Modern glassmorphism design with smooth animations
- **Accessibility**: ARIA labels and keyboard navigation support
- **Security Features**: Rate limiting and input validation
